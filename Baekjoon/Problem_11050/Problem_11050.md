## 문제
* 자연수 n과 정수 r이 주어졌을 때 이항 계수 nCr을 구하는 프로그램을 작성하시오.

## 입력
1. 첫째 줄에 n 과 r 이 주어진다. (1 ≤ n ≤ 10, 0 ≤ r ≤ n)

## 출력
 *  nCr을 출력한다.

| 예제 입력1 | 예제 출력1 | 
|---|:---:|
| <pre><code>5 2</code></pre> | <pre><code>10</code></pre> |
