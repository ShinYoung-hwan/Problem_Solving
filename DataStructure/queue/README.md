# Queue

## 큐란?
 - 원소의 추가나 삭제가 **First-in First-out(FIFO)** 구조인 집합
 - 먼저 추가된 원소가 먼저 제거됨
 - 원소의 추가는 후면에서, 원소의 제거는 전면에서 발생함.
 
## 용어정리
 * Front
    - 큐의 가장 전면부
    - 큐에서 원소의 제거가 발생하는 부분

 * Rear
    - 큐의 가장 후면부
    - 큐에서 원소의 추가가 발생하는 부분
 * Enqueue
    - Rear에 원소를 추가하는 것

 * Dequeue
    - Front에서 원소를 제거하는 것

## 기능
 * initQueue
    - 큐를 초기화한다.

 * isFull
    - 저장공간이 정해져있는 경우 큐가 가득차있는지 확인한다.

 * isEmpty
    - 큐가 비어있는지 확인한다.

 * peek
    - Front에 있는 원소를 확인한다.

 * enqueue
    - Rear에 원소를 추가한다.

 * dequeue
    - Front에서 원소를 제거한다.
